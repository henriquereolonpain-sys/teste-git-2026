##### Fluxo de trabalho Git locasl

01. git checkout -b
02. cria ou atualiza arquivos
03. git status 
04. git add arquivos
05. git status
06. git commit -m "minha mensagem"
07. git checkout main
08. git merge nova\_branch

##### Fluxo de trabalho GitHub <> Local (projeto próprio ou da sua empresa)

-git clone
-git checkout -b <nova\_branch>
-alterações de arquivos
-git status
-git add arquivos
-git status
-git commit -m "nova mensagem"
-git push origin <nova\_branch>
-abrir Pull request no GitHub para main
-excluir <nova\_branch> origin
-git checkout main
-git branch -D <nova\_branch>
