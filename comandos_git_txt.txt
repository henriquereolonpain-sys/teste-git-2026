pwd -> caminho
ls  -> opções
cat -> abre repo
cd -> abre a pasta
git init. -> cria um git, deixa especial
git add -> add o q vai entrar no check point
git status -> descrição do git
git commit -m -> criando um checkpoint e criando uma descrição 
git config -> pra atualizar name e e-mail
cd .. -> volta uma casa
git diff -> diferença do estado atual pro anterior
git reset -> tu tira faze de stage (após git.add)
git log
git branch -b -> cria uma branch nova
git merge-> une duas branches
git checkout -> muda de branch
git branch -d -> deleta branch
